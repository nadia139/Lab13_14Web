# Dashboard & Tabel Barang
Setelah login, admin akan melihat tabel data yang rapi dengan kolom berikut:
- Gambar: Menampilkan foto produk (contoh: Parfum, Sepatu, Kalung).
- Nama Barang: Nama spesifik produk.
- Kategori: Klasifikasi produk (Fashion, Kosmetik, Perhiasan, dll.).
- Harga Jual & Beli: Format mata uang dengan desimal (Rp XX.XXX,00).
- Stok: Jumlah barang yang tersedia.
- Aksi: Tombol Edit (Hijau) dan Delete (Merah).
